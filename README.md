# uhv - Community Food Sharing Web App

**uhv** is a second-year student web application designed to manage and facilitate **food sharing** within communities. Built with **Flask (Python)** and **HTML**, it provides a simple and intuitive interface for users to share, request, and track food items efficiently.

---

## Features

- Share available food items with the community  
- Request food items when needed  
- Track and manage shared food  
- Simple web interface using HTML templates  
- Dynamic routing and content management via Flask

---

## Technologies Used

- **Python** – Backend logic and routing  
- **Flask** – Web framework for handling requests and templates  
- **HTML** – Frontend structure and layout
